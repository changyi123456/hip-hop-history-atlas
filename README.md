# 嘻哈脈絡 Hip-Hop Histories Atlas

一張把 216 段中英雙語街頭故事放上黑膠與地鐵線的互動圖譜，從布朗克斯派對、MC／DJ／breaking、採樣與區域場景，一路走到臺灣、串流與 AI。

## 開啟方式

- 直接開啟根目錄的 `嘻哈歷史脈絡知識圖譜.html`
- 資料驗證：`npm run validate:data`
- 重新產生離線網頁：`npm run build`
- 完整測試：`npm test`

## 操作

- 左鍵拖曳：旋轉黑膠溝槽式圖譜
- Shift＋拖曳或右鍵拖曳：平移
- 滾輪：縮放
- 點擊節點：開啟歷史主張、證據、爭議與來源
- `/`：聚焦搜尋；`Esc`：清除搜尋或關閉詳情
- 可切換中英文與暗／亮色，偏好只保存在本機瀏覽器
- 每個節點以「先聽故事、為什麼有料、真實現場、別被帶偏」四層敘事呈現
- 黑膠本體、紐約地鐵路線與克制的街頭塗鴉介面
- 點開故事站後黑膠與整張知識圖譜會慢速旋轉，關閉詳情即停止；支援系統減少動態效果設定

研究方法、史觀與授權界線見 [RESEARCH.md](./RESEARCH.md)。
